# Sinistro — Visão geral

Nesta etapa, o objetivo é registrar e acompanhar o sinistro no ciclo:

- **SINAV**: aviso/abertura e alterações do sinistro
- **SINPEND**: saldo pendente (posição financeira do sinistro)
- **SINPAG**: pagamento efetivo (liquidação) do sinistro

Cada arquivo entra como registros **PENDING** e é processado para relacionar com apólice/certificado e com o sinistro existente.
