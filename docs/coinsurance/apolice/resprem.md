# RESPREM — Descrição negocial (Coinsurance)

> **Definição curta:** **RESPREM** é a **resposta/resumo** do que foi processado para prêmio (emissão/endosso), usado para **conciliação e retorno**.

## O que ele representa no negócio

O RESPREM costuma consolidar:

- Identificação do movimento (ramo, apólice, endosso, proposta)
- Datas de emissão e vigência
- Totais de prêmio e indicadores relevantes para reconciliar com o que foi ingerido

## Para que serve no fluxo operacional

- Funciona como “espelho de resultado” (o que o sistema reconheceu/registrou como processado).
- Suporta conciliação e auditoria do ciclo: **emitido → parcelado → recebido**.

!!! note
    O formato exato e a semântica podem variar conforme acordos operacionais/integração.
